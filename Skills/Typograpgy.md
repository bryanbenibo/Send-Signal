# typography Rules 
This document defines the typographic systems used trough this application

The system follows principles influenced by the Material Design 3 typography system

## Objective 
Typography provides:
- clear hierarcy
- consistent ready rhythm
- comfortable scanning 

## Font family 
Primary UI Font : Inter

Fallback font 
-apple-system
sysytem -UI
sanserif
arial
Roboto  

## Type scale

Use the type scale in the design tokenss

## Usage Guidelines
- Page display text should use display large 
- Section titles should use headline large
- Button labels should use label large
- Form labels should use label medium
- Standard paragraph and body text should use body large 
- Card titles should use title medium

## Accessiblity
- All text must be measured in rems not pixels
- All heading and display text must use the clamp function in css
- Low contrast text combination are avoided
- Text palce on color backgrounds must match the "on-*" color token
